# 日影观测试验记录报告

## 实验器材
- 指南针
- 木板（30cm*30cm）
- 一次性筷子 (h=11cm)

## 实验地点
陕西省图书馆思想者雕塑正后方台面上（34°13′57″N，108°56′30″E）

## 试验方案
1. 将筷子固定在木板正中央，一次性筷子处为原点，构建平面直角坐标系
2. 每天在12:10在上述地点持续观测一周（8.1-8.7）
3. 记录下每一天影子顶端的坐标
4. 总结结论
